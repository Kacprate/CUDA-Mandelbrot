# CUDA-Mandelbrot
Mandelbrot set rendered using CUDA

# Controlls:
- WSAD - move
- E - zoom in
- Q - zoom out
- R - increase cursor speed
- F - decrease cursor speed
- T - increase max iterations
- G - decrease max iterations
- I - toggle render information
